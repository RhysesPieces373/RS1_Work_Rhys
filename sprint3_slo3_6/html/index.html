sprint3_slo3_6
Sprint 3 - SLO 3.6 - Rhys Darcy 24447183

This program requires the use of the gazebo simulator and ros2. This program will detect a 30cm (0.3m)

How to Initialise//Install

To initialise sim: 

colcon build --packages-select sprint3_slo3_6

In three different terminals, enter: 

source ~/ros2_ws/install/setup.bash

In one terminal each (when there is a map in your home directory (below) and have the required launch file), enter:

Location where map should be placed (Home)
export TURTLEBOT3_MODEL=waffle_pi 
ros2 launch turtlebot3_gazebo turtlebot3_park.launch.py

ros2 run sprint3_slo3_6 cyl_det

export TURTLEBOT3_MODEL=waffle_pi 
ros2 launch turtlebot3_navigation2 navigation2.launch.py use_sim_time:=true map:=$HOME/my_map.yaml

Program functionality

This program utilises the LaserScan message from the /scan topic to detect 30cm cylinders around the turtlebot3. (You will have to create a model for the 30cm cylinder manually).
