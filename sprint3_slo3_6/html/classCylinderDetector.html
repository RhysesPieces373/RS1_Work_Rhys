Detailed Description

Cylinder Detector ROS2 Node.

This class is a ROS2 node that allows a robot (turtlebot3) to detect 30cm cylinders.

Author
Rhys Darcy
Version
1.0
Date
08/10/2024
Bug:
None found as of 08/10/2024
Warning
Will not work if gazebo and a map on RViz2 is not open

Detects a cylinder from Cartesian coordinates and returns its index.

This function searches through vectors of Cartesian points from a laser scan and groups them into continuous clusters based on how close they are. If a cluster's width is within a certain threshold of +-0.05m of the 0.3m cylinder, it is detected and its index is returned.

Parameters
[in]xsvector of Cartesian x-coordinates
[in]ysvector of Cartesian y-coordinates
Returns
Index of the cylinder in the LaserScan

Publishes a cylinder visualisation_msgs marker at (x,y) in RViz2.

This function creates and publishes a cylinder marker at (x,y), which is the detected cylinder's position. This position is also transformed from the robot's base_link frame to the map's frame.

Callback function used to process/use laser scan message data.

This function searches for cylinders in the robot's general vicinity after receiving a LaserScan through /scan. Once detected, the function will then publish a marker and broadcast a transform
